There are many ways to get started with the murals.

Check out the map and look around and find out more about the collections.

Download the data and try it out in a tool like Palladio or Carto. Palladio helps you to better understand how you might map and visualize the data for your own project. Carto allows for quick and easy mapping.

Data is provided under a Creative Commons Attribution [version 4.0] License. If you have any questions about using this data please contact me [arutkowski at library.ucla.edu].
